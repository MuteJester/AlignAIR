# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment and discriminatory behavior are not
tolerated. We want a welcoming environment for everyone contributing to or using AlignAIR.

Report unacceptable behavior to the maintainers at **thomaskon90@gmail.com**. Reports will be
handled confidentially.
