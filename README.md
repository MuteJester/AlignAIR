## Prerequisites

Before using the VDeepJ model trainer, make sure you have the following prerequisites installed:

- Python 3.8
- numpy==1.21.5
- tensorflow==2.5.0
- tensorflow-addons==0.16.1
- tensorflow-gpu==2.9.1
- pandas==1.3.5
- scipy==1.9.1
- seaborn==0.12.1
- see requirements.txt for more detailed list
